# titanic-streamlit-app
Application Streamlit interactive pour l'analyse et la prédiction de survie sur le Titanic (Projet d'apprentissage Machine Learning)

Ce projet est une application web interactive d'analyse de données et de Machine Learning utilisant le célèbre dataset du Titanic. Il a été conçu dans un objectif d'apprentissage pour maîtriser le cycle complet d'un projet Data Science : de l'exploration à la mise en production (Deployment)

#### Objectifs du Projet

    - Analyse Exploratoire (EDA) : Comprendre les facteurs influençant la survie (Sexe, Classe, Âge).

    - Visualisation : Créer des graphiques interactifs pour illustrer les tendances.

    - Modélisation : Entraîner et comparer plusieurs algorithmes de classification binaire.

    - Interactivité : Permettre à l'utilisateur de choisir ses modèles et visualiser les performances en temps réel.

#### Technologies Utilisées

    - Langage : Python 3.x

    - Interface : Streamlit

    - Manipulation de données : Pandas, Numpy

    - Visualisation : Matplotlib, Seaborn

    - Machine Learning : Scikit-Learn (Logistic Regression, SVC, Random Forest)

#### Fonctionnalités de l'Application

L'application est divisée en trois sections principales :

    - Exploration : Affichage des données brutes, dimensions du dataset et analyse des valeurs manquantes.

    - Data Visualization : Graphiques de répartition (countplots), analyses croisées et matrice de corrélation (Heatmap).

    - Modélisation : * Prétraitement automatique (Imputation des valeurs manquantes, Encodage, Standardisation).

          . Sélecteur de modèle interactif.

          . Affichage des scores de performance (Accuracy et Matrice de Confusion).
